# Codeforces Statistics Dashboard

A React app to view your Codeforces statistics including rating history, problems solved, and more.

## Features

- User stats (rating, problems solved, contests)
- Rating history graph with time filters
- Problems solved by rating distribution
- Topic-wise problem analysis
- Problem solving calendar

## Setup

```bash
npm install
npm start
```

## Usage

Enter a Codeforces handle and click "Get Stats" to see the statistics.

## Tech Stack

- React
- Recharts
- React Calendar
- Codeforces API
